## Code

```java
class Solution {
    public int rangeBitwiseAnd(int left, int right) {
        int bitCount = 0;
        while (left != right) {
            left = left >> 1;
            right = right >> 1;
            bitCount++;
        }
        return left << bitCount;
    }
}
```
